# Atlas

**Atlas** — веб-платформа для управления школьным расписанием: ручное редактирование на сетке, проверка конфликтов в реальном времени и автогенерация черновиков (CP-SAT, жадный алгоритм, локальный поиск).

```text
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│  Next.js UI │────▶│  FastAPI API │────▶│ PostgreSQL  │
│  :13000     │     │  :18080      │     │  :15432     │
└─────────────┘     └──────────────┘     └─────────────┘
       │                    ▲
       └──── Nginx :18090 ──┘  (единый origin + /api)
```

## Возможности

| Область | Что умеет Atlas |
|--------|------------------|
| **Расписание** | Drag-and-drop сетка, отмена действий, валидация перед сохранением |
| **Учебный план** | Часы по классу и предмету (`class_subject_hours`), статус покрытия плана |
| **Автогенерация** | CP-SAT по школе, жадный черновик по классу, reoptimize, GA-полировка |
| **Сценарии** | Отсутствие учителя, замена, сокращённый день, недоступный кабинет |
| **Аналитика** | Нагрузка учителей, загрузка кабинетов, качество расписания, тепловые карты |
| **Импорт** | Excel-шаблон: валидация и коммит за один проход |
| **Экспорт** | XLSX и PDF по классу, учителю или всей школе |
| **i18n** | Интерфейс и API: `kk`, `ru`, `en` |

## Быстрый старт

```bash
docker compose up --build -d
```

| Сервис | URL |
|--------|-----|
| Интерфейс (Next.js) | http://localhost:13000 |
| Через Nginx | http://localhost:18090 |
| API | http://localhost:18080 |
| Health | http://localhost:18080/health |
| Postgres (хост) | `localhost:15432` |

**Демо-аккаунты** (задаются в `docker-compose.yml`):

| Роль | Email | Пароль |
|------|-------|--------|
| Admin | `admin@atlas.example.com` | `AtlasSeed!2026` |
| School Manager | `manager@atlas.example.com` | `AtlasSeed!2026` |

Миграции и seed выполняются автоматически при старте backend (`docker-entrypoint.sh`).

## Стек

- **Frontend:** Next.js 15, React, next-intl, React DnD
- **Backend:** FastAPI, SQLAlchemy, Alembic, OR-Tools (CP-SAT, опционально)
- **БД:** PostgreSQL 16
- **Инфра:** Docker Compose, Nginx

## Вне скоупа MVP

- Многонедельная ротация как отдельный продукт
- Telegram-бот, ERP, мобильное приложение
- Интеграции с гос. платформами

См. репозиторий. Для локальной разработки и тестов используйте `docker compose` и [руководство по разработке](docs/development.md).
